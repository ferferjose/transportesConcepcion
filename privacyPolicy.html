2. Recogida de datos personales

Este sitio web tiene carácter meramente informativo. No se recogen datos personales mediante formularios, ni se almacenan datos identificativos de los visitantes. Los únicos datos que podrían ser tratados son aquellos que el usuario proporciona voluntariamente a través del correo electrónico facilitado en el apartado de contacto.

3. Finalidad del tratamiento

Los datos facilitados voluntariamente a través del correo electrónico serán tratadosv únicamente con el fin de atender las solicitudes o consultas realizadas por el usuario. No se utilizarán para envíos comerciales, boletines, ni cesiones a terceros.

4. Conservación de los datos

Los datos se conservarán únicamente durante el tiempo necesario para gestionar la comunicación iniciada por el usuario y nunca más allá del plazo legal necesario.

5. Derechos del usuario

El usuario puede ejercer sus derechos de acceso, rectificación, supresión, oposición, limitación del tratamiento o portabilidad enviando un correo electrónico a: transconcepcion18@gmail.com

6. Seguridad

Se han adoptado las medidas técnicas y organizativas necesarias para garantizar la confidencialidad, integridad y disponibilidad de los datos.

7. Cambios en la política de privacidad

Transporte Mejías se reserva el derecho de modificar esta política para adaptarla a futuras novedades legislativas o jurisprudenciales.
